# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code
of conduct.

The full text is available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

In short: be kind, be patient, assume good faith, and keep discussions
focused on the work. Discriminatory, demeaning, or hostile behavior is
not welcome.

## Reporting

If you experience or witness behavior that violates this code, please open
a private security advisory at:

<https://github.com/nicejungle/glyph8/security/advisories/new>

You may also reach the maintainer via GitHub at
[@nicejungle](https://github.com/nicejungle). Reports are handled
confidentially.

## Scope

This code applies in all project spaces (issues, pull requests,
discussions, commit messages) and when an individual is officially
representing the project in public spaces.
